# Training Planner
**Автор:** Асултанова Зухра Зелимхановна
## Описание
**Training Planner** — это графическое приложение для планирования тренировок. Оно позволяет:
- Добавлять тренировки с указанием даты, типа и длительности
- Просматривать список всех тренировок в удобной таблице
- Фильтровать тренировки по типу и по дате
- Автоматически сохранять данные в файл `trainings.json`
- Загружать сохранённые данные при следующем запуске
## Требования
- Python 3.7+
- Библиотека Tkinter (входит в стандартную поставку Python)
- Модули: `json`, `datetime`, `os` (все встроенные)
